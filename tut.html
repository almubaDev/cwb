Paso 0: Introducción

¡Bienvenido! Este tutorial te guiará paso a paso para instalar Python en tu sistema operativo Windows. Python es un lenguaje de programación versátil y popular, ideal tanto para principiantes como para desarrolladores experimentados.

Sigue las instrucciones cuidadosamente. Haz clic en "Siguiente" para comenzar.

Paso 1: Descargar el Instalador de Python

Primero, necesitas descargar el instalador oficial de Python.

  1. Abre tu navegador web y ve a la página oficial de Python: python.org/downloads/windows/
  2. Busca la última versión estable de Python para Windows. Generalmente, estará destacada en la parte superior. Haz clic en el enlace de descarga para "Windows installer (64-bit)" si tienes un sistema de 64 bits (la mayoría de los modernos), o el de 32 bits si es necesario.

(Imagen ilustrativa del logo de Python. Visita el enlace para ver la página de descargas real.)

Guarda el archivo .exe en una ubicación fácil de recordar, como tu carpeta de "Descargas".

Paso 2: Ejecutar el Instalador

Una vez descargado el archivo .exe (por ejemplo, python-3.X.Y-amd64.exe), haz doble clic sobre él para iniciar el proceso de instalación.

(Ejemplo de un archivo instalador .exe en el explorador)

Aparecerá la primera pantalla del instalador de Python.

Paso 3: Configuración Inicial (¡MUY IMPORTANTE!)

Esta es una de las pantallas más importantes. Asegúrate de marcar dos casillas clave:

  • "Install launcher for all users (recommended)": Suele estar marcada por defecto.
  • "Add Python X.Y to PATH": ¡Esta es crucial! Marcar esta opción te permitirá ejecutar Python desde cualquier parte de la línea de comandos (CMD o PowerShell) sin tener que especificar la ruta completa al ejecutable.
¡Atención! Si olvidas marcar "Add Python X.Y to PATH", tendrás que configurarlo manualmente más tarde, lo cual puede ser un poco más complicado.

(Fuente: Microsoft Docs - Instalador de Python)

Después de marcar estas casillas, puedes hacer clic en "Install Now" para una instalación estándar con las opciones recomendadas, lo cual es suficiente para la mayoría de los usuarios. Si deseas personalizar la ubicación o los componentes, puedes elegir "Customize installation". Para este tutorial, asumiremos "Install Now".

Paso 4: Proceso de Instalación

El instalador comenzará a copiar los archivos necesarios y configurar Python en tu sistema. Verás una barra de progreso.

(Fuente: ICT.social - Barra de progreso de instalación de Python)

Este proceso puede tardar unos minutos. ¡Ten paciencia!

Si aparece una ventana de Control de Cuentas de Usuario (UAC) preguntando si permites que la aplicación haga cambios en tu dispositivo, haz clic en "Sí".

Paso 5: Instalación Completada

Una vez finalizada la instalación, verás un mensaje indicando "Setup was successful".

(Fuente: Microsoft Docs - Instalación de Python exitosa)

En esta pantalla, también podrías ver una opción para "Disable path length limit". Es recomendable hacer clic en ella si aparece, ya que elimina una limitación antigua de Windows sobre la longitud máxima de las rutas de archivo, lo que puede causar problemas con algunos proyectos de Python. Si no aparece, no te preocupes.

Haz clic en "Close" para cerrar el instalador.

Paso 6: Verificar la Instalación

Para asegurarte de que Python se ha instalado correctamente y está accesible desde la línea de comandos, vamos a verificarlo.

  1. Abre una nueva ventana de Símbolo del sistema (CMD) o PowerShell. Puedes buscar "cmd" o "powershell" en el menú de inicio.
  2. Escribe el siguiente comando y presiona Enter:
    python --version

Deberías ver la versión de Python que acabas de instalar (por ejemplo, Python 3.X.Y).

(Fuente: PythonForBeginners.com - Verificar versión de Python)

También puedes verificar la instalación de pip, el gestor de paquetes de Python, que se instala automáticamente:

  1. En la misma ventana de CMD o PowerShell, escribe:
    pip --version

Deberías ver la versión de pip y la ruta donde está instalado.

(Fuente: Stack Overflow - Verificar versión de pip)

Si alguno de estos comandos no funciona (por ejemplo, dice "'python' no se reconoce..."), es probable que la opción "Add Python to PATH" no se haya marcado correctamente durante la instalación o que necesites reiniciar tu PC (o al menos cerrar y volver a abrir todas las ventanas de terminal).

Paso 7: ¡Felicidades!

¡Has instalado Python exitosamente en tu sistema Windows!

Ahora estás listo para empezar a programar en Python. Puedes ejecutar scripts, usar el intérprete interactivo (escribiendo python en la terminal y presionando Enter) o instalar paquetes adicionales usando pip.

(Fuente: Python.org - Logo oficial)

Algunos próximos pasos podrían ser:

  • Elegir un editor de código o IDE (como VS Code, PyCharm, Sublime Text).
  • Comenzar un curso de Python para principiantes.
  • Explorar la documentación oficial de Python.

¡Disfruta tu viaje con Python!
